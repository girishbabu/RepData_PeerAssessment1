Are there differences in activity patterns between weekdays and weekends?

First, let us separate our dates in weekdays and weekends so we can start splitting the interval counts among them. For this we will create a factor variable and add it to the dataset

activity.weekdays = weekdays(dt.activity$date)
factor.weekday = factor( ifelse(activity.weekdays == "Saturday" 
                                | activity.weekdays == "Sunday",
                                "weekend",
                                "weekday"),
                         levels = c("weekend","weekday"))
dt.activity[, weekstatus := factor.weekday]
